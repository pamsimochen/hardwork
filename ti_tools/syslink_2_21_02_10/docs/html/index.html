Modules

The SysLink product is partitioned into several components:

  • System APIs:
    • SysLink
  • Slave processor management APIs:
    • ProcMgr
  • Low-level IPC-related APIs
    • Ipc
    • Ipc (host only)
    • GateMP
    • HeapBufMP
    • HeapMemMP
    • ListMP
    • MessageQ
    • MultiProc
    • NameServer
    • Notify
    • SharedRegion
  • High-level data transfer APIs:
    • RingIO
  • Utility APIs:
    • Cache
    • Gate
    • List
    • Memory
  • Deprecated APIs:
    • FrameQ
